GETAWAYS PLATFORM - QUICK REFERENCE GUIDE

For Guests

Booking a Listing

  1. Search for listings
  2. Select dates and guests
  3. Review pricing
  4. Apply coupon (optional)
  5. Choose payment method
  6. Submit booking request
  7. Wait for host confirmation
  8. Complete payment (if needed)

Managing Bookings

  • View: Account Settings → Bookings
  • Cancel: Booking Details → Cancel Booking
  • Pay: Booking Details → Complete Payment
  • Review: Booking Details → Write Review

Payment Methods

  • GetPay Wallet: Instant payment
  • PayPal: PayPal integration
  • Pay Later: Reserve now, pay on confirmation

For Hosts

Creating a Listing

  1. Sign up as host
  2. Choose listing type (Accommodation/Experience/Service)
  3. Complete onboarding steps
  4. Connect PayPal
  5. Pay subscription
  6. Publish listing

Managing Bookings

  • View Requests: Host Dashboard → Bookings
  • Accept/Decline: Booking Details → Accept/Decline
  • View Calendar: Host Dashboard → Calendar

Earning Points

  • First Listing: 50 points
  • Confirmed Booking: 1 point per ₱100
  • Positive Review: 5-25 points (based on rating)

Creating Coupons

  1. Host Dashboard → Coupons
  2. Create Coupon
  3. Enter code and discount
  4. Set validity period
  5. Save coupon

For Admins

Common Tasks

Process Refund

  1. Admin Dashboard → Bookings
  2. Find cancelled booking
  3. Review cancellation
  4. Calculate refund
  5. Process refund
  6. Update status

Release Earnings

  1. Admin Dashboard → Earnings
  2. Find completed booking
  3. Verify completion
  4. Release earnings
  5. Transfer to host PayPal

Manage Users

  1. Admin Dashboard → Users
  2. Search/filter users
  3. View user details
  4. Take action (if needed)

Generate Report

  1. Admin Dashboard → Reports
  2. Select report type
  3. Choose date range
  4. Generate report
  5. Download PDF/Excel

Keyboard Shortcuts

  • Ctrl/Cmd + K: Search
  • Esc: Close modals
  • Tab: Navigate forms

Status Codes

Booking Status

  • Pending: Awaiting host confirmation
  • Confirmed: Host accepted, payment processed
  • Completed: Stay/service completed
  • Cancelled: Booking cancelled

Payment Status

  • Pending: Payment not yet processed
  • Paid: Payment completed
  • Refunded: Payment refunded

Listing Status

  • Published: Visible to guests
  • Unpublished: Hidden from search
  • Draft: Not yet published

Support Contacts

  • Email: support@getaways.com
  • Help Center: /help
  • Policies: /guest/policies or /host/policies

Quick Tips

For Guests

  • Verify email after registration
  • Read listing details carefully
  • Communicate with hosts before booking
  • Leave honest reviews
  • Check cancellation policies

For Hosts

  • Complete profile information
  • Upload high-quality photos
  • Respond to requests promptly
  • Maintain accurate calendar
  • Communicate clearly with guests

For Admins

  • Review bookings regularly
  • Process refunds promptly
  • Monitor platform activity
  • Respond to support requests
  • Maintain platform policies

Emergency Procedures

Payment Issues

  1. Check transaction history
  2. Verify payment method
  3. Contact support with booking ID
  4. Provide transaction details

Booking Conflicts

  1. Check availability calendar
  2. Contact host/guest
  3. Resolve conflict
  4. Update booking if needed

Account Issues

  1. Try password reset
  2. Check email verification
  3. Contact support
  4. Provide account details
